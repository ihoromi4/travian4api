Получение информации:

- Данные игры
    - Версия сервера (+)
    - Язык сервера (+)
    - Время сервера (+)

- Данные пользователя
    - Имя (+)
    - Нация (+)
    - Население (+)
    - Альянс (+)
    - Количество поселений (+)

- Информация о деревне
- Информация о ресурсах
- Информация о войске
- Информация о рынке
- Информация о культуре
    - Количество культуры
    - Прирост культуры
    - Необходимо для следующего поселения
- Информация о одобрении
    - В выбраной деревне
    - Во всех деревнях
- Информация о экспансии
- Информация о альянсе
    - Участники
    - Нападения
- Информация о карте
- Статистика
- Отчеты
- Сообщения


Управление учетной записью:

- Любые здания
    - Определение цены постройки (+)
    - Определение времени постройки (-)
    - Постройка (+)

- Использование главного здания
    - Определение множителя времени строительства (-)
    - Снос зданий (-)

- Использование рынка
    - Определение количества торговцев (+)
    - Отправка ресурсов (+)
    - Принятие торговой сделки (-)
    - Выставление сделки на рынок (-)
    - Использование NPC-торговца (-)

- Резиденция или дворец
    - Обучить поселенцев (-)
    - Определение количества деревень (+)
    - Определение возможного количества деревень (+)
    - Культура
        - Определение количества культуры (+)
        - Определение необходимого количества культуры (+)
        - Определение недостающего количества культуры (+)
        - Определение производство культуры в деревне (+)
        - Определение производство в остальных деревнях (+)
        - Определение производство герой (+)
        - Определение полное производство культуры (+)
    - Определение одобрения (-)

- Казарма, конюшня, мастерская
    - Определение списка возможных юнитов (-)
    - Определение цены постройки юнита (-)
    - Определение времени постройки юнита (-)
    - Определение количества юнитов на доступные ресурсы (-)
    - Постройка указаного количества юнитов (-)
    - Определение списка обучающихся юнитов (-)

- Кузница
    - Усовершенствование войск (-)

- Академия
    - Исследование войск (-)

- Пункт сбора
    - Список собственных войск (-)
    - Список подкреплений (-)
    - Список перемещающихся войск (-)
    - Определение потребления зерна войсками (-)
    - Отправка войск (-)

- Ратуша
    - Запуск маленьких праздников (+)
    - Запуск больших праздников (-)
    - Определение информации о праздниках (-)

- Отчеты
    - Удаление (-)

- Активация +25% к производству (-)
