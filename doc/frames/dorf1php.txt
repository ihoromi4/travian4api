Один из основных фреймов с которым мы встречаемся.

Информация на фрейме для любой деревне:
- Время сервера
- Количество золота
- Количество серебра
- Список деревень
- Количество отчетов
- Количество сообщений

Информация на фрейме для выбраной деревни:
- Имя деревни
- Обьем склада
- Обьем амбара
- Количество ресурсов
- Выработка ресурсов
- Свободно зерна для новых зданий
- Список ресурсных полей
- Уровни ресурсных полей
- Количество ресурсов на строительство до следующего уровня
- Отмечена постройка которая строится
- Список текущего строительства (имя, уровень, время завершения)
- Перемещение войск
- Список войск
